## Requirements
- PHP 8+
- Laravel 8+
- Filament 2+

## Description:
Creates migration, model, and filament resource with form and tables, all in one command :D

## How to import:
composer require sixtyfour-software/make-full-resource

## How to use it:
php artisan make:full-resource-64
